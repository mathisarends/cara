# Bundled audio models

- `silero_vad.onnx`: Silero VAD 6.2, downloaded from the upstream
  [`silero-vad`](https://github.com/snakers4/silero-vad) repository. MIT licensed.
  SHA-256: `1a153a22f4509e292a94e67d6f9b85e8deb25b4988682b7e174c65279d8788e3`.
- `smart_turn_v3_2_cpu.onnx`: quantized Smart Turn v3.2 CPU model, downloaded
  from [`pipecat-ai/smart-turn-v3`](https://huggingface.co/pipecat-ai/smart-turn-v3)
  at commit `f766f81d3cfdf7737ac64aad813d91bbfd56bf93`. BSD-2-Clause licensed.
  SHA-256: `2bb026316b14a660486a75b1733cd3fbab8c2fd0314dc9af7be49f8cca967e4f`.
